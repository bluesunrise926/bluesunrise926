GitHub 設定

填入您的 GitHub Token，系統將直接透過 API 將報告存入您的倉庫。Token 僅儲存在您的瀏覽器本地，不會傳送至任何第三方。

安全提醒：請確保您使用的是具備 repo 權限的 Personal Access Token（Fine-grained 或 Classic 均可）。建議為此系統單獨建立一個 Token，並設定適當的有效期限。
