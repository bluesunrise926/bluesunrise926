# 📥 統一報告輸入格式模板（Report Input Template）

> **使用說明**：每次將新報告交給 AI 分析時，請先用本模板整理報告資訊，再貼入對話框。統一的輸入格式能讓 AI 精準理解報告的性質與您的分析目標，大幅減少幻覺風險。

---

## ▌輸入品質檢查清單

在提交給 AI 之前，請確認以下事項：

| 檢查項目 | 確認 |
|----------|------|
| 報告來源與日期已填寫 | ☐ |
| 摘要內容是我自己整理的，非直接複製 AI 輸出 | ☐ |
| 關鍵數據已填入對應表格 | ☐ |
| 我的具體問題已明確寫出 | ☐ |
| 已標注相關的現有記憶庫條目 | ☐ |
| 可信度評估已填寫 | ☐ |
